Common CSS Layout Methods

Floats

Originally for floating images inside blocks of text, the float property became one of the most commonly used tools for creating multiple-column layouts on webpages. With the advent of Flexbox and Grid, it has now returned to its original purpose.

Positioning

Positioning allows elements to be taken out of normal document flow and behave differently, such as overlapping or staying fixed in the browser viewport.

Flexbox

Flexbox is a one-dimensional layout method used for arranging items in rows or columns. Items flex to fill available space and shrink to fit smaller screens.

CSS Grid Layout

CSS Grid Layout is a two-dimensional system that organizes content into rows and columns, simplifying the creation of complex layouts.
